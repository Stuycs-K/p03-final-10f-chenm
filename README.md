### Project Description:

This is a multiplayer blackjack game where the computer is the dealer. You and others can join a game session and play blackjack. A score will show after each round to show each player's scores.

### Instructions:

- User has to type in "make compile" and "./server" to start the server. Players can join with "./client"
- Client just has to follow the instructions by typing in hit/stand to play the game.
