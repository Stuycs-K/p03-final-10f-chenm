# Blackjack

### Michelle's Casino

Michelle Chen

### Project Description:

This is a multiplayer blackjack game where the computer is the dealer. You and others can join a game session and play blackjack but you can't see the other player's hand. You can only see the dealer's hand and you can type hit or stand to play the game. Everyone has to finish for the server to proceed and thne it will tell you the score.

### Video:
https://drive.google.com/file/d/1EMe6CIJ83RYrl4r5sOIxQoP30_HWK1KP/view?usp=sharing

### Instructions:

- User has to type in "make compile" and "./server" to start the server. Players can join with "./client"
- Client just has to follow the instructions by typing in hit/stand to play the game.
